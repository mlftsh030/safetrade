# SafeTradesa
## Safetradesa is a trust-focused platform designed to enhance safety in peer-to-peer online marketplaces such as Facebook Marketplace.

The platform addresses critical challenges such as fraud, fake listings, and unsafe in-person transactions by introducing features like user verification, secure transaction flows, and trust scoring.

This project follows a data-driven development approach, beginning with market validation through user research and behavioral analysis, and evolving into a scalable application aimed at restoring confidence in digital trading environments.

TrustTrade demonstrates the integration of product thinking, data analytics, and full-stack software engineering to solve real-world problems in emerging digital economies.
